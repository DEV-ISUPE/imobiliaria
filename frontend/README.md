# Maison Huambo - Frontend

Landing page e painel administrativo da imobiliária Maison Huambo.

## Estrutura

- `index.html` - Página inicial com catálogo de imóveis
- `admin.html` - Painel administrativo do corretor
- `css/` - Folhas de estilo
- `js/` - Lógica assíncrona com Fetch API
- `assets/` - Imagens e ícones

## Setup

Abra os arquivos HTML no navegador ou use um servidor local:

```bash
# Usando Python
python3 -m http.server 8080

# Usando Node
npx serve
```

O frontend consome a API em `http://localhost:3000/api`.
